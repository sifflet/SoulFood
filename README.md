## Controls

#### Collector

| Key Binding | Action |
| ----------- | ------ |
| Arrow Keys | Move player along x-axis and z-axis |
| Left Shift | Drop item |
| Left Control | Pick up item |

#### Guard

| Key Binding | Action |
| ----------- | ------ |
| W, A, S, D | Move Guard #1 along x-axis and z-axis |
| Arrow Keys | Move Guard #2 along x-axis and z-axis |
